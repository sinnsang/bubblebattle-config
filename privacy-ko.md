# BubbleBattle 개인정보처리방침

**최종 업데이트: 2026-07-06**

개인 개발자(이하 "당사")은 모바일 게임 **BubbleBattle**("본 앱")을 이용하는 사용자의 개인정보를 소중히 다루며, 관련 법령을 준수합니다. 본 방침은 본 앱이 수집·이용하는 정보와 사용자의 권리를 안내합니다.

## 1. 수집하는 정보

본 앱은 다음의 정보를 최소한으로 수집합니다.

| 항목 | 수집 시점 | 저장 위치 | 목적 |
|---|---|---|---|
| **광고 식별자** (IDFA/AAID) | 앱 실행 시 (광고 SDK 자동 수집) | Google AdMob 서버 | 맞춤형 광고 노출, 광고 노출 빈도 관리 |
| **기기 고유 식별자**(랜덤 생성 문자열) | 최초 실행 시 자동 생성 | 기기 내부(iOS Keychain / Android SharedPreferences) | 멀티플레이어 세션 재접속 |
| **플레이어 닉네임** | 멀티플레이 방 참여 시 사용자가 직접 입력 | Firebase Realtime Database (인메모리 방 상태) | 다른 참여자에게 표시 |
| **익명 인증 UID** | 앱 부팅 시 Firebase 익명 인증 | Firebase Authentication | 서버 접근 권한(익명), 방 접근 제어 |
| **게임 진행 정보**(HP, 위치 좌표, 처치 수 등) | 멀티플레이 진행 중 | Firebase Realtime Database | 실시간 게임 동기화 |

**당사가 수집하지 않는 정보**: 이름, 이메일, 전화번호, 실제 위치정보(GPS), 사진, 연락처, 결제 정보 등 민감 정보를 일절 수집하지 않습니다.

## 2. 이용 목적

수집된 정보는 다음 목적으로만 이용됩니다.

- 멀티플레이어 매칭 및 실시간 게임 상태 동기화
- 광고 노출 및 광고 서비스 개선 (Google AdMob)
- 부정 이용 방지, 통계 분석을 통한 서비스 개선
- 앱 버전 관리 및 강제 업데이트 안내

## 3. 보유 및 이용 기간

- **멀티플레이 방 데이터**: 방 생성 후 최대 6시간, 또는 마지막 참여자 이탈 시 자동 삭제
- **기기 고유 식별자**: 앱 삭제 시까지 (재설치 시 iOS는 유지·Android는 재생성)
- **광고 식별자**: 사용자가 기기 설정에서 리셋/차단 시 즉시 반영
- **익명 인증 UID**: Firebase 정책에 따라 사용자가 재설치 없이 계속 이용 시 유지

## 4. 제3자 제공 및 위탁

당사는 서비스 제공을 위해 다음 사업자에게 정보 처리를 위탁합니다.

| 위탁 대상 | 용도 | 제공 정보 | 개인정보처리방침 |
|---|---|---|---|
| **Google LLC — Firebase** | 멀티플레이 세션 저장·인증 | 익명 UID, 플레이어 닉네임, 게임 상태 | https://firebase.google.com/support/privacy |
| **Google LLC — AdMob** | 광고 게재 | 광고 식별자, 기기 정보 | https://policies.google.com/technologies/ads |

당사는 위 목적 외에 사용자 정보를 판매·양도하지 않습니다.

## 5. 사용자 권리 및 삭제 요청

- **광고 개인화 거부**: 기기 설정 → 개인정보 보호 → Apple 광고(iOS) / Android 광고(Android) 에서 광고 ID 리셋/차단
- **데이터 삭제**: 앱 삭제 시 기기 내부 데이터(닉네임 저장 값 등) 즉시 제거
- **문의 및 요청**: 아래 이메일로 요청 시 7영업일 이내 처리

## 6. 아동 개인정보 보호

본 앱은 만 13세 미만 아동을 주요 대상으로 하지 않습니다. 만 13세 미만 아동의 정보가 수집된 사실을 인지한 경우 즉시 삭제합니다. 아동이 앱을 이용하실 경우 보호자의 감독 하에 이용하시기 바랍니다.
